Technical Term Glossary
Term Definition Image
CSS "Cascading Style Sheet" - Formats the layout of web pages. They can be used to define text styles & edit positioning.
HTTP "Hypertext Transfer Protocol" - The protocol used to transfer data over the web.
JavaScript A client side scripting language that controls the behaviour of a web page.
SMTP "Simple Transfer Mail Protocol" - It is used for sending emails over the internet.
SSL "Secure Sockets Layer" - A secure protocol used for sending data and information securely over the internet.
SVG "Scalable Vector Graphic" - Graphics file that uses a 2D vector graphic format.
